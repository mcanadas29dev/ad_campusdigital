<h2> Ojo importante (esto suele caer en examen) </h2>
<ul>
    <li>Regresión Lineal → solo para valores numéricos (NO Iris)</li>
    <li>Regresión Logística → clasificación (SÍ Iris)</li>
    <li>Árbol de Decisión → clasificación (SÍ Iris)</li>

</ul>

<h2>Traducción directa: </h2>
<ul>
    <li>Si te dan Iris → usa Logística o Árbol</li>
    <li>Si te dan precios, temperaturas, etc. → Regresión Lineal</li>
</ul>



<h2>Recordar</h2>
<p>Separar → dividir → entrenar → predecir → evaluar”</p>
