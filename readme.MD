<h2> Ojo importante (esto suele caer en examen) </h2>
Regresión Lineal → solo para valores numéricos (NO Iris)
Regresión Logística → clasificación (SÍ Iris)
Árbol de Decisión → clasificación (SÍ Iris)

<h2>Traducción directa: </h2>

Si te dan Iris → usa Logística o Árbol
Si te dan precios, temperaturas, etc. → Regresión Lineal

<h2>Recordar</h2>
